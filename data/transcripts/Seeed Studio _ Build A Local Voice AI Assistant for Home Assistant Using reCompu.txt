We built a Local Voice AI Assistant using the Seeed reComputer RK3576 and reSpeaker XVF3800 microphone array to enable a fully on-device, prompt-based Home Assistant control workflow powered by:

✅ Local wake-word / prompt-based activation
✅ Whisper-based speech recognition (STT)
✅ Local LLM for intent understanding
✅ mms_tts for local text-to-speech generation

Now the system can:
👂 Capture and understand natural voice commands
🧠 Interpret prompts for Home Assistant control
🤖 Respond with real-time synthesized voice feedback
⚡ Run fully offline with NPU-accelerated inference
